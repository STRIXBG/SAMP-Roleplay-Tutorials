Стъпки:
1. Изтеглете си папката за Windows(SA-MP 0.3.7-R2 Windows Server) или тази за Linux(SA-MP 0.3.7-R2 Linux Server (x86)
) от www.sa-mp.com
2. Поставете я където желаете и добавете папките, които са в този Tutorial
3. За Windows: пуснете сървъра, като кликнете върху samp-server.exe
   За Linux: от секцията на сайта на хостъра, натиснете START SERVER (не във всички хостинг сайтове е с такова име!)
4. Готово, вашият сървър вече е онлайн
5. Ако все още не го намирате в търсачката, напишете :7777 (Windows Hosting от вашия компютър)
   Или за Linux: IP-то го пише задължително на сайта на вашия хоситнг.
